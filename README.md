# FEM Lost & Found

Created by **Taemin Park (MYP 4)** for Service as Action.

## Tech Stack
- **Flask** for the backend  
- **SQLite** for the database  
- **Jinja**, **HTML**, and **CSS** for the frontend  
- **Pillow** for image uploading  
- **Gunicorn** for deployment and hosting
